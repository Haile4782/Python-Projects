This repository contains a collection of five introductory Python programming projects designed to practice core concepts like loops, conditional statements, data structures (lists and dictionaries), and user input/output.

---

## Projects Included

### 1. Vending Machine Program
A command-line program that simulates a vending machine transaction, tracking selected items, prices, and generating a final receipt.
*Skills: loops, input, conditionals, dictionaries.*

### 2. Simple Grocery Cart Checkout
A text-based checkout system using a predefined price dictionary. It handles quantities, calculates subtotals, and displays a formatted final bill.
*Skills: dictionaries, error handling, formatting.*

### 3. To-Do List App (Text-Based)
A basic task manager that allows users to add, view, delete, and mark tasks as complete, showing a summary upon exit.
*Skills: lists, string parsing, CRUD basics.*

### 4. Movie Ticket Booking Simulation
A simulator that displays movie options, calculates the total cost for a chosen number of tickets, and handles multiple bookings.
*Skills: nested dictionaries, calculations, loops.*

### 5. Quiz Game
A simple quiz application that loads questions, tracks the user's score, and provides correct answers for missed questions at the end.
*Skills: dictionaries, comparison, counters.*
